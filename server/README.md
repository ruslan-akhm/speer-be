### Run locally

Make sure to add .env file to the **server** folder.
.env suppose to have variables:

- DATABASE
- TOKEN

```sh
$ git clone https://github.com/ruslan-akhm/speer-be.git
$ cd speer-be
$ cd server
$ npm install
$ npm start
```

For testing run

```sh
$ npm test
```
